Create your own board.

By clicking "New Board" button, you can create your own board. In this page, you can iew all public/your own/closed boards.

Create a new list.

By clicking "New List" button, you can create a new list, in which you can add cards.

Add a new card.

By clicking "Add a card" button, you can add cards in any list. (You have had the ermission to edit this board.)

Set the board to a private board.

By clicking this icon, you can set the board to a private board.

Invite others to board.

You can invite other people into the board. Thus they have permissions to edit this oard.

You can view all activities of this board.
